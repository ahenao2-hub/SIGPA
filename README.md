## 📋 Descripción del Proyecto

SIGPA es un software que reemplaza el manejo disperso de  
hojas de cálculo, correos eléctronicos y carpetas físicas por un sistema centralizado que permite:

- Gestionar los **8 niveles de práctica pedagógica**
- Asignar plazas a estudiantes de forma eficiente
- Cargar evidencias y bitácoras en tiempo real
- Generar reportes
